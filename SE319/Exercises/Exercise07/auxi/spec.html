Your Assignment

Assume that x.html is the html file which includes your hw5.Fall12.js javascript file and contains a two-level list element as per the above specification. You are required to write javascript functions in the hw5.Fall12.js such that the following dynamics are realized in x.html (for your assignment, we have provided one such x.html in the form of hw5.Fall12.html):
  1. Intialize: When the x.html is loaded, a pop-up window stating "Hello! I am <yournetid>" should show up, and only the top-level list items should be visible
  2. Expand: Clicking on a top-level list item makes the second level list items visible. Nothing else in view of the x.html should change.
  3. Collapse: Clicking on a already expanded top-level list item makes its second level list items invisible. Nothing else in the view of the x.html should change.
  4. Explore: Clicking on a second level list item will open the corresponding html page in the section identified by "id=content" in the x.html page.
Do not update the given html and css files.
